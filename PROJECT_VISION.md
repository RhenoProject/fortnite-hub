## Mission

フォトナHub を Fortnite ユーザーにとって最も便利な総合情報プラットフォームにする。

収益化をし収益を伸ばしていくこと。

---

## North Star

**Fortnite の情報を探すなら最初に フォトナHub を開く状態を作る。**

---

## Vision（3年後）

- 日本最大のフォートナイト情報プラットフォーム
- 月間100万PV
- 複数の収益源（AdSense・アフィリエイト・スポンサーシップ）
- しゅうやCEO の主要収入源の一つ

---

## Value Proposition

| ユーザー | 提供価値 |
|---------|---------|
| カジュアルプレイヤー | 毎日のショップ・ニュースを1サイトで確認 |
| 競技プレイヤー | 最新アップデート・競技日程を即座に確認 |
| ガジェット購入検討者 | プロ仕様のおすすめデバイスを参照 |

---

## Target User

- 日本国内の Fortnite プレイヤー
- 10代〜30代
- スマートフォンメインで情報収集

---

## Differentiators（競合との差別化）

1. **日本語特化** — 英語サイトが多い中、日本語で完結
2. **毎日自動更新** — 手動更新なしで常に最新情報
3. **シンプルUI** — 余計な機能を排除した高速サイト
4. **通知機能** — ほしいスキンが出たら即プッシュ通知

## Success Metrics

### Primary KPI
- Organic Traffic（検索流入）
- Returning Users（リピート率）
- Search Usage（サイト内検索利用率）
- User Retention（継続訪問）

### Secondary KPI
- CTR
- Session Duration
- Page Views
- Social Shares

## Constraints（制約）

- 人間リソース: しゅうやCEO 1名のみ
- 予算: 最小限（Vercel無料枠・Firebase無料枠）
- 技術: Next.js + Vercel + GitHub のみで完結
- 時間: 自動化を最大化し人力作業を最小化
