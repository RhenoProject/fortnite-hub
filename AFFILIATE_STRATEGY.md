# AFFILIATE_STRATEGY.md

> 責任者: AI CTO ジョブズ（兼 Affiliate事業責任者）
> 最終更新: 2026-06-19（初版）

---

## Mission

Fortnite Hubの長期的な価値の最大化を最終目的とする。

アフィリエイト収益の最大化ではない。

---

## Core Principle

**ユーザー価値 → 信頼 → 収益** の順番を守る。

収益を優先してユーザー体験を犠牲にしてはならない。

---

## Fortnite Hubの中核（不変）

* Fortnite情報
* Fortniteニュース
* Fortniteデータ
* Fortniteユーザーの課題解決

アフィリエイトはこの価値を補完する手段。目的ではない。

---

## 取り扱いルール

### 対象カテゴリ（Fortniteユーザーと関連性が高い）

| カテゴリ | 関連性 | 優先度 |
|---------|-------|-------|
| ゲーミングデバイス（マウス・キーボード・ヘッドセット） | 高 | 最優先 |
| ゲーミングモニター | 高 | 最優先 |
| コントローラー | 高 | 最優先 |
| ゲーミングチェア | 中〜高 | 優先 |
| PC・GPU | 中 | 検討 |
| ゲーミングイヤホン | 高 | 優先 |

### 取り扱わない商品

* Fortniteユーザーへの価値が低い商品（収益性が高くても除外）
* Fortnite Hubのブランドと関連性が低い商品
* 短期的な収益目的だけで選定した商品

---

## コンテンツ優先順位

**商品ページ単体より以下を優先:**

1. **比較ページ** — 「マウスAとマウスBどっちがいい？」を解決
2. **ランキングページ** — 「フォートナイト向けおすすめマウス TOP5」
3. **購入ガイド** — 「フォートナイト初心者向けデバイス選び方」
4. 商品ページ（上記の補完として）

---

## KPI（計測指標）

| 指標 | 意味 |
|------|------|
| Affiliate Revenue | 実収益 |
| CTR | クリック率（商品リンクのクリック÷ページPV） |
| Product Clicks | リンクの総クリック数 |
| Conversion Opportunities | 購入につながる接触機会 |
| Commercial Intent | 購買意図のあるユーザー比率 |

---

## 新規コンテンツ・カテゴリの評価基準

提案時は以下5点を必ず評価する:

| 項目 | 説明 |
|------|------|
| User Value | Fortniteユーザーの課題を解決するか |
| Search Demand | 検索需要（月間検索数）があるか |
| Revenue Potential | 収益化できるか（単価・転換率） |
| Competition | 競合が弱い・勝てる余地があるか |
| Implementation Cost | 実装コストはどれくらいか |

ROIが最も高い施策を最優先する。アイデア数より期待インパクトを重視。

---

## 実行ルール

* 商品ページ作成前に比較・ランキング・購入ガイドで課題解決できないか必ず検討する
* 収益だけを目的にしたコンテンツは作成しない
* 短期的収益より長期的信頼とブランド価値を優先する
* CEOがリンクを提供した後の作業は最大限自動化する
* 必要なリンクは一覧化してCEOへ要求する
